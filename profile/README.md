# ottabase

**ottabase** is a framework built for solo SaaS founders — Cloudflare-first and batteries included, so you can ship fast without the overhead.

- ⚡ **Cloudflare-first** — built to run on the edge with Cloudflare Workers, Pages, D1, R2, and more
- 🔋 **Batteries included** — auth, billing, database, storage, email, and everything else you need to launch a SaaS, ready to go
- 🧑‍💻 **Built for solo founders** — sensible defaults, minimal boilerplate, and a fast path from idea to production

🌐 [ottabase.com](https://ottabase.com)
